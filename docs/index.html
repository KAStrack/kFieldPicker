A zero-dependency pattern picker for any text input.
#@/{{ Trigger on any character. Filter as you type.

Attach after init decoupled from tinymce.init
Wiring the picker doesn't have to live inside tinymce.init's setup. This attaches it separately and even before init finishes — it waits on the EditorManager (tinymce.get / AddEditor / the editor's init event), so a slow CDN load can't race it.

11
Template trigger #value#
Triggered by #, inserts a #value# token via insertTemplate — pick cat and #cat# is inserted. closeChar: '#' makes the picker treat the trigger as a symmetric delimiter, so a completed #cat# is recognised as closed. Multi-word items work too: #sea o keeps sea otter showing and inserts #sea otter#.
